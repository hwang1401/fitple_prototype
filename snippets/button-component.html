Primary button
페이지 구성 시 위계가 가장 높은 버튼이에요.
CTA button
본래는 사용자의 클릭을 유도하는 버튼으로 정의되지만 에너지 모니터링 시스템 혹은 플랫폼 등을 운영하는 자사의 특성상 해당 버튼은 사용자 권한에 따라 특정 기능에 대한 사용가능 여부를 매핑하기 위한 버튼이에요. 물론 기존 CTA가 가진 사용법으로도 사용 가능해요.
Secondary button
강조 버튼을 보조하는 역할을 하는 버튼이에요. 보통 primary나 CTA에 버튼과 조합하여 사용해요.
Link button
완전히 다른 화면으로 전환되거나 새 창이 별도의 탭이나 창으로 실행되는 경우에 사용해요.
Size
버튼의 크기는 높이값만 조정해요. xlg, lg, md, sm, xsm 등의 사이즈가 있으며 기본적으로 md 크기를 사용해요.
